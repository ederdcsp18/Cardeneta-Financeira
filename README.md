Cardeneta Financeira - AC3 Mobile Development
Este repositório contém o projeto desenvolvido para a Atividade Continuada 3 (AC3) da disciplina de Mobile Development. A aplicação tem como objetivo principal o gerenciamento de finanças pessoais, permitindo o controle de entradas e saídas de forma simplificada.

📋 Descrição do Projeto
O Cardeneta Financeira é um aplicativo mobile que auxilia o usuário a manter o histórico de suas movimentações financeiras. O foco desta etapa (AC3) foi a implementação de fluxos de navegação, persistência de dados e uma interface intuitiva seguindo os padrões de design mobile.

🚀 Funcionalidades Implementadas
Registro de Transações: Cadastro de receitas e despesas com descrição e valor.

Visualização de Saldo: Cálculo automático do balanço total com base nas entradas e saídas.

Listagem Dinâmica: Exibição das movimentações de forma organizada.

Interface Responsiva: Adaptada para diferentes tamanhos de tela.

🛠️ Tecnologias Utilizadas
Framework: Flutter

Linguagem: Dart

Persistência de Dados: SQLite / Shared Preferences

Gerenciamento de Estado: Provider / setState

🔧 Como executar
Certifique-se de ter o Flutter instalado e configurado em sua máquina.

Clone este repositório:

Bash
git clone https://github.com/ederdcsp18/Cardeneta-Financeira.git
Acesse a pasta do projeto:

Bash
cd Cardeneta-Financeira
Instale as dependências:

Bash
flutter pub get
Execute o aplicativo:

Bash
flutter run
